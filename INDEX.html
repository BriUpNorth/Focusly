Measuring Pelvic Obliquity

What is pelvic obliquity?

Pelvic obliquity is the lateral tilt of the pelvis in the coronal (frontal) plane — one side of the pelvis is higher than the other. It is measured as the angle between the line connecting the two anterior superior iliac spines (ASIS) and the horizontal.

```

Equipment needed

  • Inclinometer or goniometer
  • Ruler or tape measure (for indirect method)
  • Flat surface / plinth

Method 1 — Inclinometer (preferred)

1. Position the patient supine on a firm, level surface.

2. Locate both ASIS points and mark them.

3. Place the inclinometer flat across both ASIS marks (or along the line connecting them).

4. Read the angle directly from the inclinometer. Zero = level; positive values = obliquity present.

5. Note which side is elevated (R High or L High).

Method 2 — Tape measure (indirect)

1. Position the patient supine on a level surface.

2. Measure the vertical height from the plinth to each ASIS using a tape measure held perpendicular to the surface.

3. Record both measurements in cm.

4. Calculate the height difference between sides.

5. Use a calculator or reference chart to convert height difference and inter-ASIS distance into degrees, or record the height difference in cm as a proxy measure.

Scoring reference

0–10° → 0 points — within typical range
11–20° → 1 point — mild obliquity
>20° → 2 points — significant obliquity

Clinical notes

Pelvic obliquity may be fixed (structural) or flexible (postural). If obliquity reduces or resolves when the knees are repositioned, it is likely to have a flexible component — record both the initial angle and the improved angle in Section 2B.

PCS Windsweeping Profile™

```

Overview

The PCS Windsweeping Profile™ evaluates patterns of lower‑limb rotational asymmetry and how they influence pelvic, hip, knee, and ribcage alignment. It provides a structured method for identifying windsweeping tendencies without using any copyrighted systems.

1 — Resting Limb Position (RLP) Score max 3

Record resting rotation angles (in degrees):

  • Right resting rotation angle (°)
  • Left resting rotation angle (°)
  • Difference = |Right – Left|
0–10° → 0 points
11–25° → 1 point
26–40° → 2 points
>40° → 3 points

2 — Hip Rotation Mobility (HRM) Score max 3

Assess hip internal/external rotation range.

Within 10° symmetry → 0 points
15–25° difference → 1 point
>25° difference → 2 points
Rotation difference with discomfort / resistance → 3 points

3 — Knee Tracking & Influence (KTI) Score max 3

Observe knee movement during flexion/extension.

Symmetrical tracking → 0 points
Mild drift (in/out) → 1 point
Moderate drift affecting trunk/pelvis → 2 points
Strong drift or rigidity → 3 points

4 — Foot & Ankle Alignment Effect (FAA) Score max 2

Record foot posture and rotation.

Within 10° symmetry → 0 points
10–25° asymmetry → 1 point
>25° asymmetry or foot drives rotational bias → 2 points
PCS Windsweeping Profile Total
RLP + HRM + KTI + FAA
Maximum 11 points

Interpretation Bands

0–2 → No windsweeping pattern
3–5 → Mild windsweeping tendency
6–8 → Moderate windsweeping pattern
9–11 → Significant windsweeping posture
